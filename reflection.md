I gradually developed this project using Git commits the aim of my project was to create a solution to a problem many face on a daily basis, many vulnerable people fall for email scams regularly, I have created a scam message detector where the user input the senders email, subject of email, and the email body. The program will give each a score and decide how likely the message is a scam and give reasons. I started with the EmailMessage class, then i implemented a base Rule class, I then further expanded this to the UrgencyRule, SenderRule and LinkRule. Furthermore I created the ScamDetector class to combine all rules and calculate a final score. I used regex to search for patterns in text, in lines for example 'urgent: ,your account will be deleted' regex finds 'urgent', I used word boundaries (\b) to ensure that full words match. I used the built in python 're' library to let pattern matching take place, which allowed scam-related keywords to be searched and URLs to be extracted from users messages. My use of OOP was highlighted by the use of classes, for example 'EmailMessage' which stores email data, it helped me structure my project into resuable and modular components. I further implemented inheritance by creating subclasses such as UrgencyRule, SenderRule and LinkRule.
